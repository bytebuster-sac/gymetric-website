Módulos

Control de Asistencias
  • Registro de entradas mediante lector biométrico.
  • Historial de asistencia por usuario.
Área de Planes y pagos
  • Crear y editar planes de membresía (duración, precios).
  • Asignación automática de planes y control de vencimientos.
Registro de Clientes
  • Base de datos completa.
  • Búsqueda y filtrado por ID, nombre o plan para acceso rápido.
Gestión de Membresías
  • Renovación y vencimiento automático
  • Gestión de cambios de plan y control de estados (activo, suspendido, vencido).
Reportes
  • Informes de ingresos y vencimientos por periodo.
  • Reportes de asistencia y uso por cliente o por plan.
Usuarios y roles
  • Control de accesos con roles (admin, recepcionista, entrenador).
  • Permisos por módulo para limitar y auditar acciones del personal.

Preguntas frecuentes

¿Qué es Gymetric?

Gymetric es un sistema de control de asistencia biométrica que utiliza tecnología de huellas digitales para gestionar el acceso y la seguridad en establecimientos.

¿Cómo funciona el sistema biométrico?

El sistema utiliza lectores de huellas digitales para registrar y verificar la identidad de los usuarios, permitiendo un control de acceso seguro y eficiente.

¿Qué medidas de seguridad tienen mis datos?

Utilizamos encriptación avanzada y protocolos de seguridad para proteger toda la información almacenada en nuestra plataforma.
